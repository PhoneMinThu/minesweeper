# minesweeper

A simple terminal Minesweeper game written in Rust.

## Requirements
- Rust toolchain (via rustup)

## Build
```
cargo build
```

## Run
```
cargo run
```

## Controls
- Arrow keys or WASD to move cursor
- Space or Enter to reveal a cell
- F to flag/unflag a cell
- Q to quit
